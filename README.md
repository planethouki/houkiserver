# houkiserver

A minecraft server.

# Environment

* CentOS 7
* Using Spigot
* Directory `/var/minecraft/spigot`
* BackupDir `/var/minecraft/backup`
* DynmapDir `/var/www/html/minecraft/dynmap`
* Run User `minecraft`
